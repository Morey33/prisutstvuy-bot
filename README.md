# @prisutstvuy_bot

Тихий проводник по вниманию и состоянию.

## Запуск локально

1. Скопируй `.env.example` в `.env` и заполни переменные
2. Установи зависимости: `pip install -r requirements.txt`
3. Запусти: `cd bot && python main.py`

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен от @BotFather |
| `OPENAI_API_KEY` | Ключ OpenAI для классификации состояний |
| `ADMIN_TELEGRAM_ID` | Твой Telegram ID (узнать у @userinfobot) |
| `DB_PATH` | Путь к SQLite файлу (только локально) |
| `DATABASE_URL` | URL PostgreSQL (на Railway автоматически) |

## Деплой на Railway

1. Подключи этот репозиторий в Railway
2. Добавь переменные окружения в настройках
3. Railway запустит бота автоматически

## Команды

- `/start` — начало
- `/gift` — получить подписку бесплатно
- `/grant <id>` — открыть доступ пользователю (только админ)
- `/cancel` — отменить текущее действие
